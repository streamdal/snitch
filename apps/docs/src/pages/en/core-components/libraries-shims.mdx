Libraries, shims, wrappers, or extensions are libraries that incorporate the
usage of our SDKs to simplify the integration and adoption of Streamdal.

Shims are intended to be "drop-in" replacements for existing library usage.

<Notification type="note">
    <img src={tip} alt="Note" width="30px" height="30px" />
    <div class="prose-md prose-p:my-2 prose-a:text-purple-dark prose-a:underline">
        <p style="font-weight: lighter;">For example, if your Go application is using [segmentio/kafka-go](https://github.com/segmentio/kafka-go),
        then you can just change your import line from `github.com/segmentio/kafka-go`
        and change it to `github.com/streamdal/segmentio-kafka-go`.</p>
        <p style="font-weight: lighter; font-style: italic;">With the shim, you no longer will need to _manually_ instrument every bit of code that reads or writes data to Kafka.</p>
    </div>
</Notification>
